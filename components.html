ComponentsÂ¶

LASIF is organized into multiple components which handle various things. The standard user should not need to interact with these components because the API and the CLI should cover all the needed functionalities to run a full-waveform inversion.

However, if anyone wants to do some more advanced scripting, one can interact with the components of LASIF directly. In order to do so, one would need to access the LASIF communicator object and with that, everything should be possible. The LASIF communicator object is specific to each project and can be accessed via the API interface.
